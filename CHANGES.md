# Release notes — 0.1.3

A "Try it" trial now boots in the language the visitor is reading the
catalogue in.

Moodle ships only English strings; every other language is a pack the
sandbox downloads as it boots. The trial blueprint now asks for the
visitor's own interface language, so a teacher browsing the Exchange in
Japanese gets a Japanese sandbox. English visitors are unaffected and pay no
extra download, and a pack that fails to download costs nothing — the trial
simply comes up in English.
